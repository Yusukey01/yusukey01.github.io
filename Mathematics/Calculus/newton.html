Line Search

In the previous chapter, we used a "fixed" learning rate(or step size), which was chosen based on experimental results. However, the "optimal" step size can be determined by line search: \[ \eta_t = \arg \min _{\eta > 0} \mathcal{L} (\theta_t + \eta d_t). \] Consider a quadratic loss: \[ \mathcal{L}(\theta + \eta d) = \frac{1}{2}(\theta+ \eta d)^T A (\theta+ \eta d) + b^T (\theta + \eta d) + c. \] Taking derivative of this function: \[ \begin{align*} \frac{d \mathcal{L}(\theta + \eta d)}{d\eta} &= d^T A(\theta + \eta d) + d^T b \\\\ &= d^T(A\theta + b) + \eta d^T A d \end{align*} \] Setting the derivative equal to zero, we have: \[ \eta = - \frac{d^T(A\theta + b)}{d^T A d}. \]
