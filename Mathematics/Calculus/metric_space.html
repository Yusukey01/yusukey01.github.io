Introduction

Up until this chapter, we relied on calculus and linear algebra to solve optimization problems. For many practical machine learning applications, these tools are sufficient. However, as we move toward advanced topics like Manifold Learning or Information Geometry, intuition alone is no longer enough.

We need to be able to formally answer questions such as: "Does this optimization algorithm truly converge?" or "How do we define the structure of a space when we treat entire functions as single points?" To address these, we must shift from concrete computation to Mathematical Analysis. Concepts like completeness and topology are not just theoretical abstractions; they are the formal requirements that ensure our optimization paths actually lead to a solution.

Metric Space

In Section I, we defined metric space as following:

Metric Space \((M, d)\): Suppose \(M\) is a set and \(d\) is a real function defined on the Cartesian product \(M \times M\). Then \(d\) is called a distance function, or metric on \(M\) if and only if \(\forall a, b, c \in M\),
  1. Positivity: \(d(a,b) \geq 0\) with equality iff \(a = b\),
  2. Symmetry: \(d(a,b) = d(b,a)\), and
  3. The triangle inequality \(d(a,b) \leq d(a,c) + d(c, b)\).
