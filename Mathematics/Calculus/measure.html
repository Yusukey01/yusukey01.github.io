Riemann Integrals

First, let's quickly review one of the most familiar integral, Riemann Integrals.

Let \(f: [a, b] \to \mathbb{R} \) be a bounded function defined on a closed interval \([a, b]\) where \(a, b \in \mathbb{R}\).

A partition \(\mathcal{P}\) of \([a, b]\) is a finite sequence of points: \[ \mathcal{P} = \{x_0, x_1, x_2, \cdots, x_n\} \] where \(a = x_0 < x_1 < x_2 < \cdots < x_n = b\) and \(n \in \mathbb{N}\).

The collection of all subintervals that are formed by \(\mathcal{P}\) are given by: \[ [x_{i-1}, x_i] \text{for } i = 1, 2, \cdots, n. \]
For each subinterval, the infimum (greatest lower bound) of \(f\) is denoted by: \[ m_i = \inf_{x\in [x_{i-1}, x_i]} f(x) \] and the supremum (least upper bound) of \(f\) is denoted by: \[ M_i = \sup_{x\in [x_{i-1}, x_i]} f(x). \]
Here, we define the lower sum: \[ L(f, \mathcal{P}) = \sum_{i=1}^n m_i (x_i - x_{i-1}) \] and the upper sum: \[ U(f, \mathcal{P}) = \sum_{i=1}^n M_i (x_i - x_{i-1}). \]
The lower Riemann integral of \(f\) over \([a, b]\) is \[ \underline{\int_a^b} f(x)dx = \sup_{\mathcal{P}} L(f, \mathcal{P}) \] and the upper Riemann integral of \(f\) over \([a, b]\) is \[ \overline{\int_a^b} f(x)dx = \inf_{\mathcal{P}} U(f, \mathcal{P}). \]
A function \(f\) is said to be Riemann integrable on \([a, b]\) if \[ \underline{\int_a^b} f(x)dx = \overline{\int_a^b} f(x)dx = \alpha. \] When this holds, the common value \(\alpha\) is called the Riemann integral of \(f\) over \([a, b]\): \[ \int_a^b f(x)dx = \alpha. \]
Note: Equivalently, we can use the norm of the partition \(\mathcal{P}\): \[ \| \mathcal{P} \| = \max_{1 \leq i \leq n} (x_i - x_{i-1}). \] then a function \(f\) is said to be Riemann integrable on \([a, b]\) if for any \(\| \mathcal{P} \|\), \[ \lim_{ \| \mathcal{P} \| \to 0} L(f, \mathcal{P}) = \lim_{ \| \mathcal{P} \| \to 0} U(f, \mathcal{P}) = \alpha. \]
Some Important facts
  • If a function \(f\) is Riemann integrable on \([a, b]\), then \(f\) is bounded on \([a, b]\).
  • A continuous function on \([a, b]\) is Riemann integrable on \([a, b]\).
  • A monotonic function on \([a, b]\) is Riemann integrable on \([a, b]\).

Improper Riemann Integrals

Even if a function is not Riemann integrable, we have another familiar integral, improper Riemann integral. Fortunately, it covers a large portion of integrals encountered in statistics and machine learning, especially for functions defined over unbounded domains or with singularities.

Case 1: One side of the interval is unbounded
Let \(f: [a, \infty) \to \mathbb{R} \) be a function. \(f\) is improperly integrable as follows: \[ \int_a^{\infty} f(x)dx = \lim_{b \to \infty} \int_a^b f(x)dx. \] if the limit exists and is finite. Similarly, for \(f: (-\infty, b] \to \mathbb{R} \), \[ \int_{-\infty}^b f(x)dx = \lim_{a \to -\infty} \int_a^b f(x)dx. \]
Case 2: Both sides of the interval is unbounded For a function \(f: \mathbb{R} \to \mathbb{R}\), \[ \int_{-\infty}^{\infty} f(x)dx = \lim_{a \to -\infty}\lim_{b \to \infty} \int_a^b f(x)dx \] if the double limit exists and is finite.
Note: A classic example is the Gaussian integral: \(\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}\).

Case 3: Discontinuous somewhere on the interval if a function \(f\) has a singularity at \(c \in (a, b)\), meaning it is not defined or diverges to infinity at \(c\), the improper integral is defined as: \[ \int_a^b f(x)dx = \lim_{\epsilon \to 0^+} \left(\int_a^{c-\epsilon}f(x)dx + \int_{c+\epsilon}^b f(x)dx \right) \] Similarly, if the singularity occurs at an endpoint of the interval, we define: \[ \int_a^b f(x)dx = \lim_{\epsilon \to 0^+} \int_{\epsilon}^b f(x)dx \] if \(f\) has a singularity at \(a\), or \[ \int_a^b f(x)dx = \lim_{\epsilon \to 0^+} \int_a^{b - \epsilon} f(x)dx \] if \(f\) has a singularity at \(b\).
