Riemann Integral

Surprisingly, whenever we are calculating integrations for an expected value, entropy, or KL divergence etc in statistics, or of course, in machine learning, we are implicitly performing a Lebesgue integration that is NOT what we learned in basic calculus classes. First, we revisite the familiar integral.

Let \(f: [a, b] \to \mathbb{R} \) be a bounded function defined on a closed interval \([a, b]\) where \(a, b \in \mathbb{R}\).

A partition \(\mathcal{P}\) of \([a, b]\) is a finite sequence of points: \[ \mathcal{P} = \{x_0, x_1, x_2, \cdots, x_n\} \] where \(a = x_0 < x_1 < x_2 < \cdots < x_n = b\) and \(n \in \mathbb{N}\).

The collection of all subintervals that are formed by \(\mathcal{P}\) are given by: \[ [x_{i-1}, x_i] \text{for } i = 1, 2, \cdots, n. \]
For each subinterval, the infimum (greatest lower bound) of \(f\) is denoted by: \[ m_i = \inf_{x\in [x_{i-1}, x_i]} f(x) \] and the supremum (least upper bound) of \(f\) is denoted by: \[ M_i = \sup_{x\in [x_{i-1}, x_i]} f(x). \]
Here, we define the lower sum: \[ L(f, \mathcal{P}) = \sum_{i=1}^n m_i (x_i - x_{i-1}) \] and the upper sum: \[ U(f, \mathcal{P}) = \sum_{i=1}^n M_i (x_i - x_{i-1}). \]
The lower Riemann integral of \(f\) over \([a, b]\) is \[ \underline{\int_a^b} f(x)dx = \sup_{\mathcal{P}} L(f, \mathcal{P}) \] and the upper Riemann integral of \(f\) over \([a, b]\) is \[ \overline{\int_a^b} f(x)dx = \inf_{\mathcal{P}} U(f, \mathcal{P}). \]
A function \(f\) is said to be Riemann integrable on \([a, b]\) if \[ \underline{\int_a^b} f(x)dx = \overline{\int_a^b} f(x)dx = \alpha. \] When this holds, the common value \(\alpha\) is called the Riemann integral of \(f\) over \([a, b]\): \[ \int_a^b f(x)dx = \alpha. \]
Note: Equivalently, we can use the norm of the partition \(\mathcal{P}\): \[ \| \mathcal{P} \| = \max_{1 \leq i \leq n} (x_i - x_{i-1}). \] then a function \(f\) is said to be Riemann integrable on \([a, b]\) if for any \(\| \mathcal{P} \|\), \[ \lim_{ \| \mathcal{P} \| \to 0} L(f, \mathcal{P}) = \lim_{ \| \mathcal{P} \| \to 0} U(f, \mathcal{P}) = \alpha. \]

Improper Riemann Integral

Even if a function is not Riemann integrable, we have another familiar integral, improper Riemann integral.

Case 1: \[ \int_a^{\infty} f(x)dx = \lim_{b \to \infty} \int_a^b f(x)dx \] \[ \int_{-\infty}^b f(x)dx = \lim_{a \to -\infty} \int_a^b f(x)dx \]
Case 2: \[ \int_{-\infty}^{\infty} f(x)dx = \lim_{a \to -\infty}\lim_{b \to \infty} \int_a^b f(x)dx \]
For example, we have seen the Gaussian integral: \(\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}\)
Case 3: \[ \int_a^b f(x)dx = \lim_{\epsilon \to 0^+} \left(\int_a^{c-\epsilon}f(x)dx + \int_{c+\epsilon}^b f(x)dx \right) \] if \(f\) has a singularity at \(c \in (a, b)\) and the limit exists.

Case 4: \[ \int_a^b f(x)dx = \lim_{\epsilon \to 0^+} \int_{\epsilon}^b f(x)dx \] \[ \int_a^b f(x)dx = \lim_{\epsilon \to 0^+} \int_a^{b - \epsilon} f(x)dx \]
