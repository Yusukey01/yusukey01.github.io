Neural Networks Demo

This interactive demo showcases how a simple neural network can learn to classify non-linear patterns. You can generate datasets, tweak model parameters, and visualize the training process.

How It Works

  • Model Architecture:
    • 2 input nodes (x₁, x₂)
    • 1 hidden layer with ReLU activation (adjustable number of neurons)
    • 1 output node using a sigmoid function for binary classification
  • Forward Pass: The network computes predictions using linear transformations and non-linear activations. Selecting a demo point reveals the complete step-by-step calculation.
  • Training: The network is trained using gradient descent with backpropagation to minimize binary cross-entropy loss. Techniques used for training stability include:
    • Dynamic learning rate adjustment
    • Gradient clipping
    • Early stopping
    • L2 regularization
  • Visualizations:
    • Colored data points for training and test sets
    • Decision boundary (green) showing where the prediction is 0.5
    • Probability contours representing prediction confidence
    • Network graph that updates dynamically in demo mode

Try Adjusting

  • Hidden Units: More neurons allow for more complex decision boundaries.
  • Regularization (λ): Helps prevent overfitting by discouraging large weights.
  • Learning Rate: Controls how quickly the model updates during training.
  • Max Iterations: Determines how long the model trains.
