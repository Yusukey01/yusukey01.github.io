Binary Logistic Regression

Consider discriminative classification model: \[ p(y \mid x, \theta) \] where \(x \in \mathbb{R}^D\) is an input vector, \(y \in \{1, \cdots, C\}\) is the class label, and \(\theta\) are the parameters.

When \(C = 2\), it is known as binary logistic regression. The mode can be represented by: \[ p(y \mid x, \theta) = \text{Ber }(y \mid \sigma(w^\top x + b)) \] where \(\sigma\) is the sigmoid(logistic) function, \(w \in \mathbb{R}^D\) are the weights, \(b \in mathbb{R}\) is a bias, and \(\theta = (w, b)\) are all the parameters.

Thus, \[ p(y = 1 \mid x, \theta) = \sigma(a) = \frac{1}{1 + e^{-a}} \] where \(a\) is called the logit(or the pre-activation): \[ a = w^\top x + b = \log (\frac{p}{1-p}), \quad p = p(y = 1 \mid x, \theta). \]

The sigmoid function measures the probability that the class label is \(y =1\). So, we want to set a decision rule such that \(y =1\) if and only if the class 1 is more likely than class 0. In other worlds, \[ \begin{align*} f(x) &= \mathbb{I}(p(y=1 \mid x) > p(y=0 \mid x)) \\\\ &= \mathbb{I} \left( \log \frac{p(y = 1 \mid x)}{p(y =0 \mid x)} > 0 \right) \\\\ &= \mathbb{I}(a). \end{align*} \] Therefore, the prediction function can be written as following: \[ f(x ; \theta) = b + w^\top x. \] Since \(w^\top x\) is the inner product between the weight vector \(w\) and the feature vector \(x\), in the end this function determines a linear hyperplane with normal vector \(w\) and an offset \(b\) from the origin. In other words, intuitively, the weight vector \(w\) defines the "orientation" of the decision boundary and its norm \( \| w \| \) represents the confidence of the model prediction because it affects the "steepness" of the sigmoid function.

The negative log likelihood (NNL) of \(w\) is given by: \[ \begin{align*} \text{NLL }(w) &= - \frac{1}{N} \log p(\mathcal{D} \mid w) \\\\ &= - \frac{1}{N} \log \prod_{n =1}^N \text{Ber }(y_n \mid \mu_n) \\\\ &= - \frac{1}{N} \sum_{n=1}^N \log \left[ \mu_n^{y_n} (1-\mu_n)^{1-y_n} \right] \\\\ &= - \frac{1}{N} \sum_{n=1}^N \left[ y_n \log \mu_n + (1-y_n) \log (1-\mu_n) \right] \\\\ &= - \frac{1}{N} \sum_{n=1}^N \mathbb{H}_{ce} (y_n , \mu_n) \end{align*} \] where \(\mu_n = \sigma(a_n) = \sigma(w^\top x_n)\) is the probability of class 1, and \(\mathbb{H}_{ce}(y_n, \mu_n)\) is the binary cross entropy bethween \(y_n\) and \(\mu_n\).

By some gradient-based optimization (or in practice, automatic differentiation methods), we solve: \[ \nabla_w \text{NLL }(w) = 0 \] to obtain the MLE. \[ \begin{align*} &\nabla_w \text{NLL }(w) &= - \frac{1}{N} \sum_{n=1}^N \left[ y_n (1-\mu_n) x_n - (1-y_n) \mu_n x_n \right] \\\\ &= \frac{1}{N} \sum_{n=1}^N (\mu_n - y_n) x_n, \\\\ \end{align*} \] Equivalently, in matrix form: \[ \nabla_w \text{NLL }(w) = \frac{1}{N} \left( \mathbb{1}_N^\top (\text{diag }(\mu - y)X)\right)^\top. \] where \(X \in \mathbb{R}^{N \times D}\) is the design matrix.

It is important to check if the Hessian, \(H(w)\) is positive semi-definite or not. Remember, it guarantees the convexity of the objective and thus the stationary point is the global optimum. \[ \begin{align*} H(w) &=\nabla_w \nabla_w^\top \text{NLL }(w) \\\\ &= \frac{1}{N} \sum_{n=1}^N (\mu_n( 1 - \mu_n)x_n) x_n^\top \\\\ &= \frac{1}{N} X^\top S X \end{align*} \] where \(S = \text{diag }(\mu_1(1-\mu_1), \cdots, \mu_N(1-\mu_N))\).

\(H(w)\) is positive semi-definite becuase for any nonzero vector \(v\), \[ v^\top X^\top S X v = (v^\top X^\top S^{\frac{1}{2}})(S^{\frac{1}{2}} X v) \] and then \[ \| v^\top X^\top S^{\frac{1}{2}} \|_2^2 > 0. \]

Note: Often instead of \(y \in \{0, 1\}\), we use the labels \(\tilde{y} \in \{-1, 1\}\). In this case, \[ \sigma(-a) = 1 - \sigma(a) \] and then: \[ p(\tilde{y} \mid x, \theta) = \sigma(\tilde{y}a). \]
